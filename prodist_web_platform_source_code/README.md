# ProDist Web Platform Source Code

This directory contains the source code of the ProDist web platform for deployment-oriented Vietnamese traditional-instrument recognition.

The platform demonstrates the image-only inference setting used by ProDist. It receives an input image, applies the required preprocessing steps, loads the exported classification-only student model, and returns the predicted instrument class through a web interface.

## Purpose

The web platform is provided as a deployment artifact for the ProDist study. It supports practical inspection of how the compact student model can be used in a cultural heritage expert-system setting.
